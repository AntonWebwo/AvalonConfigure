# AvalonConfigure
AvalonConfigure — это консольное приложение на C++ для настройки майнинг-устройств серии Avalon через HTTP-запросы к веб-интерфейсу устройства. Программа позволяет отправлять конфигурационные параметры (пулы, воркеры, режим работы и т. д.) на устройство по указанному IP-адресу с использованием авторизации.
